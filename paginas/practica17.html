17. Vistas (VIEWS)

Una vista (VIEW) es una consulta SQL guardada con un nombre en la base de datos. Actúa como una tabla virtual que puedes consultar como si fuera una tabla real, simplificando consultas complejas y reutilizando lógica.

¿Para qué sirven las Vistas?

Simplifican consultas largas, restringen el acceso a columnas específicas y permiten reutilizar lógica de negocio sin escribir el mismo código repetidamente.

Base de Datos: Empleados y Departamentos
1Vista simple con filtro (WHERE)

Crea una vista que filtra solo los empleados del departamento 2 (TI):

CREATE VIEW empleados_ti AS
            SELECT nombre, depto_id
            FROM empleados
            WHERE depto_id = 2;

Consultas sobre la vista:

SELECT * FROM empleados_ti;
SELECT nombre FROM empleados_ti
            WHERE nombre LIKE "%a";
2Vista con JOIN entre tablas

Crea una vista combinando empleados y departamentos con INNER JOIN:

CREATE VIEW empleados2 AS
            SELECT e.nombre AS empleado, 
                e.salario, 
                d.nombre AS departamento
            FROM empleados e 
            INNER JOIN departamentos d ON (d.id = e.depto_id);

Consultas sobre la vista:

SELECT * FROM empleados2;
SELECT departamento, COUNT(*) AS Cuantos
            FROM empleados2
            GROUP BY departamento;
Consultas Directas con JOIN (Sin Vista)

Consulta con alias Nombre_Empleado:

SELECT e.nombre AS Nombre_Empleado, e.Salario, d.nombre
            FROM empleados e 
            INNER JOIN departamentos d ON (d.id = e.depto_id);

Consulta con alias Empleado y departamento:

SELECT e.nombre AS Empleado, e.Salario, d.nombre AS departamento
            FROM empleados e 
            INNER JOIN departamentos d ON (d.id = e.depto_id);
Base de Datos: Escuelas / Alumnos
3Vista de Alumnos - Datos Académicos

Crea una vista con nombre, promedio y edad del alumno:

CREATE VIEW Alumno_2 AS
            SELECT Nombre, Promedio, Edad
            FROM alumno;

Consultas sobre la vista:

SELECT * FROM Alumno_2;
SELECT * FROM Alumno_2
            WHERE Promedio < 9;
4Vista de Alumnos - Datos Personales

Crea una vista con los apellidos y estatura del alumno:

CREATE VIEW Alumno_3 AS
            SELECT ApellidoPaterno, ApellidoMaterno, Estatura
            FROM alumno;

Consultas sobre la vista:

SELECT * FROM Alumno_3;
SELECT ApellidoPaterno, ApellidoMaterno
            FROM Alumno_3
            WHERE ApellidoPaterno LIKE "%a";
