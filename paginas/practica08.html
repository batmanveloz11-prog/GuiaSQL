8. UPDATE - Actualizar Datos (Katanas Edition)

La sentencia UPDATE en MySQL se utiliza para modificar los valores de los registros ya existentes en una tabla.

¡CUIDADO AL USAR UPDATE!

Si olvidas poner la cláusula WHERE, actualizarás todos los registros de la tabla. Verifica primero con un SELECT:

SELECT * FROM katanas WHERE [condición];
1Actualizar un solo campo por ID
UPDATE katanas
            SET precio = 1350.00
            WHERE id = 1;
2Actualizar un campo de texto por nombre
UPDATE katanas
            SET material = 'Tamahagane Refinado'
            WHERE nombre = 'Wado Ichimonji';
3Actualizar varios campos a la vez
UPDATE katanas
            SET precio = 1800.00, longitud = 76
            WHERE id = 5;
4Actualizar varios registros según una condición

Incrementar un 10% el precio de todas las katanas:

UPDATE katanas
            SET precio = precio * 1.10
            WHERE tipo = 'Katana';
5Actualizar con varias condiciones (AND)
UPDATE katanas
            SET forjador = 'Maestro Kozaburo'
            WHERE forjador = 'Kozaburo'
            AND longitud >= 73;
6Actualizar utilizando la lista IN
UPDATE katanas
            SET precio = 999.99
            WHERE id IN (3, 7);
