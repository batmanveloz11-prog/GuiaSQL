6. INSERT - Insertar Datos

Aprende los métodos esenciales para ingresar registros en una base de datos MySQL, desde inserciones simples hasta la importación de archivos CSV creados con el Bloc de notas.

Estructura de la Tabla de Prueba

Para los siguientes ejemplos, asumiremos que tenemos una tabla llamada usuarios creada con la siguiente estructura:

CREATE TABLE usuarios (
                id INT AUTO_INCREMENT PRIMARY KEY,
                nombre VARCHAR(100),
                email VARCHAR(100),
                edad INT
            );
Método 1: Inserción Individual Estándar

Es la forma básica de insertar un solo registro especificando las columnas y los valores correspondientes.

INSERT INTO usuarios (nombre, email, edad) 
            VALUES ('Juan Pérez', 'juan.perez@email.com', 28);
Método 2: Inserción Múltiple o Masiva (Batch Insert)

Permite insertar varias filas en una sola instrucción SQL, lo cual es mucho más eficiente que hacer múltiples consultas individuales.

INSERT INTO usuarios (nombre, email, edad) 
            VALUES 
            ('Ana Gómez', 'ana.gomez@email.com', 32),
            ('Carlos Ruiz', 'carlos.ruiz@email.com', 24),
            ('María López', 'maria.lopez@email.com', 19);
Método 3: Inserción desde otra Tabla (INSERT INTO ... SELECT)

Se utiliza para copiar registros de una tabla a otra de manera automatizada. Por ejemplo, transferir datos desde una tabla temporal llamada nuevos_usuarios:

INSERT INTO usuarios (nombre, email, edad)
            SELECT nombre, email, edad 
            FROM nuevos_usuarios 
            WHERE edad >= 18;
Método 4: Importación desde un archivo CSV (Bloc de Notas)

Ideal para cargar grandes cantidades de datos almacenados en un archivo de texto plano de manera rápida y directa.

Paso 1: Crear el archivo en el Bloc de Notas (Notepad)
  1. Abre el Bloc de notas.
  2. Escribe tus datos separados por comas, un registro por línea. La primera línea es la cabecera con el nombre de las columnas:
nombre,email,edad
            Laura Torres,laura.torres@email.com,26
            Pedro Marmol,pedro.marmol@email.com,40
            Sofía Castro,sofia.castro@email.com,31
  1. Guarda el archivo como datos.csv. Asegúrate de cambiar el Tipo a Todos los archivos (*.*) para evitar que se guarde como .csv.txt. Guárdalo en una ruta como C:/datos/datos.csv.
Paso 2: Ejecutar el comando SQL en MySQL

Usa la instrucción LOAD DATA INFILE para importar el archivo CSV a tu tabla:

LOAD DATA INFILE 'C:/datos/datos.csv'
            INTO TABLE usuarios
            FIELDS TERMINATED BY ',' 
            OPTIONALLY ENCLOSED BY '"'
            LINES TERMINATED BY '\r\n'
            IGNORE 1 LINES;
Nota de Seguridad: Si obtienes el error secure_file_priv, significa que tu servidor MySQL tiene restringido desde dónde puede leer archivos. Puedes usar la alternativa LOAD DATA LOCAL INFILE o colocar el archivo en el directorio permitido que te muestre el comando SHOW VARIABLES LIKE "secure_file_priv";.
