9. SELECT Básico (Katanas Edition)

La sentencia SELECT se utiliza para consultar y recuperar datos de una o más tablas en una base de datos MySQL.

1Seleccionar todos los registros
SELECT * FROM katanas;
2Seleccionar columnas específicas
SELECT nombre, precio FROM katanas;
3Filtrar registros con WHERE
SELECT * FROM katanas WHERE material = 'Tamahagane';
4Ordenar registros
SELECT * FROM katanas ORDER BY precio DESC;
5Contar registros
SELECT COUNT(*) FROM katanas;
6Buscar por ID
SELECT * FROM katanas WHERE id = 1;
7Mostrar registros únicos (DISTINCT)
SELECT DISTINCT forjador FROM katanas;
